# Ecolink - Version Beta
Application de gestion et recyclage des déchets.

## Structure du dépôt
- `backend/`: Logique métier et base de données.
- `frontend-web/`: Interface Recycleurs [cite: 127] et Statistiques[cite: 4].
- `mobile/`: Interface Ménages [cite: 70] et Collecteurs[cite: 71].
- `ai-service/`: Module d'analyse d'image par IA[cite: 51, 59].

## Workflow Git
- Branche `main`: Production.
- Branche `develop`: Intégration.
- Branches `feature/name`: Développements spécifiques.
